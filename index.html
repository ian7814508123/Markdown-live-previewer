Markdown Live Previewer - 專業線上編輯與預覽工具

Markdown Live Previewer 是一款基於 React 開發的高效能線上 Markdown 編輯器。支持即時同步預覽、Mermaid 圖表渲染、MathJax 數學公式與化學方程式、Vega-Lite 數據圖表、以及 WikiLinks 雙向連結功能。

無論您是工程師、學生還是作家，都能透過本工具輕鬆撰寫並導出高品質的 PDF、PNG、SVG 或 Markdown 檔案。

  • 即時同步滾動預覽
  • Mermaid 流程圖、時序圖、甘特圖
  • LaTeX 數學公式與化學模型支持
  • 專業 PDF/圖片 導出
  • 單機存儲，私密且快速
  • 支援 WikiLinks 雙向連結與知識管理
  • 內建 PDF 合併工具與 Excel 轉 Markdown 表格功能
  • 支援 Vega-Lite 專業數據視覺化圖表
