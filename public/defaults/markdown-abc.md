# 🎵 音樂樂譜 (abc notation)
您可以直接在 Markdown 文件中使用 `abc` 語法區塊來撰寫樂譜。本應用程式支援 `abcjs` 引擎。

## 範例音樂
```abc
X:1
T:Piano Concerto No. 2, Op. 18 - II. Adagio sostenuto
C:Sergei Rachmaninoff
M:4/4
L:1/8
Q:1/4=54
K:E
V:1 treble
%%MIDI program 73 % Flute
| z4 z2 G2 | G4- G F/G/ (A/G/F/E/) | (G4 F2) B,2 | B,4- B,A, (C/B,/A,/G,/) | (B,4 A,2) B,2 |
| (E4 D2) E2 | (G4 F2) G2 | (B4 A2) G2 | (F6) G2 | G4- GF/G/ (A/G/F/E/) |
```

> **提示**：將滑鼠懸停在樂譜上，可以透過彈出的控制列調整樂譜顯示的大小 (Scale) 與寬度 (Width)。如果您切換至深色模式，樂譜也會自動配合！
