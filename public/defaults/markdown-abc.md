# 🎵 音樂樂譜 (abc notation)
您可以直接在 Markdown 文件中使用 `abc` 語法區塊來撰寫樂譜。本應用程式支援 `abcjs` 引擎。

## Cooley's Reel (愛爾蘭傳統樂曲)
```abc
X: 1
T: Cooley's
M: 4/4
L: 1/8
R: reel
K: Emin
|:D2|EB{c}BA B2 EB|~B2 AB dBAG|FDAD BDAD|FDAD dAFD|
EB{c}BA B2 EB|B2 AB defg|afe^c dBAF|DEFD E2:|
|:gf|eB B2 efge|eB B2 gedB|A2 FA dAFA|A2 FA defg|
eB B2 eBgB|eB B2 defg|afe^c dBAF|DEFD E2:|
```

## 基礎音階

```abc
X: 2
T: C Major Scale
M: 4/4
L: 1/4
K: C
C D E F | G A B c | c B A G | F E D C |]
```

> **提示**：將滑鼠懸停在樂譜上，可以透過彈出的控制列調整樂譜顯示的大小 (Scale) 與寬度 (Width)。如果您切換至深色模式，樂譜也會自動配合！
